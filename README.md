# Robotic Arm Kinematics & Path Planning
### Project 1 | DecodeLabs Industrial Training Kit | Batch 2026

## Project Overview
A simulated 6-axis robotic arm that moves accurately between target coordinates using Inverse Kinematics, with real-time collision avoidance and smooth trajectory generation.

## Objectives
- Implement Inverse Kinematics (IK) to calculate joint angles for target XYZ coordinates
- Generate smooth, collision-free trajectories using spline interpolation
- Visualize the robot arm in 3D using ROS 2 and RViz
- Detect and avoid obstacles in real time

## Tech Stack
- **ROS 2 Humble** — Robot middleware
- **Python 3** — Control logic
- **RViz** — 3D visualization
- **URDF** — Robot model definition
